# TEAM-TECH-TRONICLES

A responsive website developed for our team, **Team Tech-Chronicles**, built using **HTML, CSS, and JavaScript**. The site is fully responsive across mobile, tablet, desktop, and smart display devices.  

---

## Features

- Fully responsive design for all devices  
- Team showcase and member information  
- Modern layout and styling using CSS  
- Interactive components powered by JavaScript
